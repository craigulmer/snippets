# Snippets

This is an aggregate repo that holds snippets of code and data from random
home projects I've work on over the years.

- **250720_cursor_bouncy_ball**: First experiment w/ Cursor. Generated a pygame with bouncy balls
- **250221_planes_datasette**: Example of how to convert airplane tracks to a SQLite DB for Datasette viewing
- **191227_googleplus_to_orgmode**: Parser to convert old Google+ posts to Emacs Org mode
- **160602_craigslist_scraper**: Scraper for pulling job postings from Craigslist
- **151114_sudsolv**: Simple Sudoku solver in C
- **151019_dem_plotters**: Digital Elevation Model plotters
- **141114_rungen**: Generate a fake run to fool data trackers
- **140908_gut_buster**: Go code for counting how frequent offensive terms appear in Project Guttenberg books
- **130330_call_cpp_from_go**: Example of how to call Cpp code from Go
